Dataset Selection

The dataset selected for this project is Gaming_Academic_Performance.csv. This dataset is suitable because it contains both gaming-related variables and academic performance measures. It also includes several numerical variables that can be used for statistical analysis and predictive modeling.

Correlation Analysis

For the correlation analysis gaming_hours and grades were used. The Pearson correlation coefficient was calculated.

Prediction Model

A Linear Regression model was chosen because the grades variable is numerical.
